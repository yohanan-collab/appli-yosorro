Application réalisée dans le cadre d'une formation ANGULAR 12h à distance pour l'école EPITA.

Formateur : Frédéric LOSSIGNOL ,
IT Teacher front-end web & mobile JS / Angular / Ionic / Flutter

Objectif : Appréhender le framework Angular et ses concepts dans le cadre du développement d'une application web.

Création d'une application type "Trello Like" :

  • Mise en place de l'environnement
  • le CLI (Command Line Interface)
  • Les ressources : module, component, service, directive, pipe,
  • tour dans une application Angular
  • Les concepts : components, data-binding, directives, isolation CSS
  • Les directives structurelles
  • L'évènement
  • Les 3 façon de passer des data entre les components
  • Architecture : Input et Output pour communiquer entre les components parents/enfants
  • Architecture : MVC : Mise en place d'un service et des méthodes accesseurs
  • Les Subjects / BehaviorSubjects pour réagir aux changements
  • Communiquer avec le monde extérieurs : faire des requests AJAX avec la librairie HtppClient
  • Mettre en place la navigation / routing
  • Bonus : Utilisation des components de la librairie Angular Material
  • Build et publication online
